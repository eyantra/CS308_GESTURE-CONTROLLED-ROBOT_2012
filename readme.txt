Project Description
-------------------
	Ours is a gesture controlled robot with gripper extension. The firebird and gripper are controlled using gestures given by the user.
	The kinect captures the gestures. 
	The laptop sends appropriate signals to firebird over zigbee module. 
	The firebird now processes these signals and takes appropriate actions
	There are 3 modes of gestures: long range control of firebird, fine control(proclainer type gestures), gripper control gestures.
